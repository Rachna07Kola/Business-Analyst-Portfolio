Projects

Malnutrition Analytics Dashboard

This project involved building an interactive Tableau dashboard analyzing malnutrition data across multiple countries over a 20-year span. I integrated data from UNICEF and WHO databases, cleaned it using Excel and SQL, and used calculated fields to identify trends in child stunting, underweight prevalence, and food insecurity. The dashboard enabled users to filter by region and year, offering a clear visualization of malnutrition hotspots and their correlation with socio-economic indicators.

Agentic AI Resume Screener

This was a cutting-edge solution where I collaborated on building a GPT-powered resume screener that mimicked human-like reasoning for shortlisting candidates. Using LangChain and OpenAI APIs, we enabled context-aware filtering of resumes against job descriptions. I developed prompt templates, fine-tuned model parameters, and deployed a prototype for internal HR review. This project demonstrated the use of agentic AI for unbiased hiring and reduced screening time by 60% in testing.

Banking Fraud Detection Using SHAP & XGBoost

Developed an AI-based fraud detection system to monitor transactions in real-time, identify anomalies, and minimize financial risk. Improved fraud detection accuracy and compliance while integrating explainability for regulatory transparency.

Tools Used: Python, ML (Supervised & Unsupervised), SHAP/LIME, AES-256, GDPR/PCI DSS standards.
